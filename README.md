For Ubuntu User

go to command prompt and type following command to run that project:
1.  cd /var/www/htdocs/AngularJsCRUD
2. npm install
3. ng serve
4. open that URL for type step 3. command run that project

For Window User

go to command prompt and type following command to run that project:
1.  cd C://xampp/htdocs/AngularJsCRUD
2. npm install
3. ng serve
4. open that URL for type step 3. command run that project
